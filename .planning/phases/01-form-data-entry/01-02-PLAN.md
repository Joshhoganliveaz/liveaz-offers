  <action>
Read the existing index.html created by Plan 01. Add the offer card JavaScript and wire the Add Offer button. Specifically:

JavaScript Functions to Add (in the script block):

1. createOfferCard():
   - Increments a monotonic offerIdCounter (never reused, used for data-offer-id attribute)
   - Creates a div.offer-card with data-offer-id attribute
   - Sets innerHTML with all field groups (detailed below)
   - Appends to #offers-container
   - Calls renumberOffers() to set display numbers
   - Returns the card element
   - Remove the placeholder message from offers-container if present

2. removeOffer(btn):
   - Gets the closest .offer-card
   - Gets the display number from the h3 text
   - Calls window.confirm('Remove Offer N? This can\'t be undone.')
   - If confirmed: card.remove(), then renumberOffers()
   - If only one card remains after removal, do not allow removal (disable or hide remove button on last card, or show a message)

3. renumberOffers():
   - querySelectorAll('.offer-card').forEach((card, i) => update h3 text to "Offer {i+1}")
   - Also show/hide remove buttons: if only 1 card, hide its remove button. If 2+, show all remove buttons.

4. Wire the Add Offer button (already in DOM from Plan 01) to call createOfferCard()

5. On DOMContentLoaded (add to existing listener): call createOfferCard() once to pre-load one empty offer card

6. Update the beforeunload guard to also check offer card inputs for content

Offer Card HTML Template (inside createOfferCard):

Each card has class="offer-card" with a subtle left border (3px solid Gold #C9953E).

Header: div.offer-header with h3 "Offer N" and a remove button (type="button", class="btn-remove", onclick="removeOffer(this)", text "Remove")

Field Group 1 - Buyer Info (OFFR-01, OFFR-07):
- Buyer Name(s): text input, full width, placeholder "First Last and First Last"

- Unrepresented Buyer: checkbox with label "Unrepresented Buyer"
Use class="field-group" with h4 "Buyer Info"

Field Group 2 - Financial Terms (OFFR-02):
- Purchase Price: number input with $ prefix
- Down Payment: number input with $ prefix
- Down Payment %: number input with % suffix (step 0.1)
- Loan Type: select with options: Select..., Conventional, FHA, VA, Cash, DSCR, Other
- Earnest Money: number input with $ prefix
Use class="field-group" with h4 "Financial Terms"

Field Group 3 - Timeline (OFFR-03):
- Close of Escrow: date input

- Offer Expiration: date input
Use class="field-group" with h4 "Timeline"

Field Group 4 - Compensation (OFFR-04):

Field Group 6 - Special Terms (OFFR-06):
