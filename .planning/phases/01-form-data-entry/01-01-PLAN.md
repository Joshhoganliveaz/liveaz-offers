Cost Template JavaScript:
- const STORAGE_KEY = 'liveaz-offer-costs'
- saveCostTemplate(): reads listing-commission, title-escrow, home-warranty, tc-fee values plus all custom items (label + amount pairs). Serializes to JSON and saves to localStorage. Called on every input event on cost fields. Wrap localStorage.setItem in try/catch.
- loadCostTemplate(): on DOMContentLoaded, reads localStorage, parses JSON, populates the four fixed fields, recreates any saved custom items by calling addCustomItem() for each. Wrap in try/catch.
- resetCostTemplate(): window.confirm('Reset all cost template values? This clears your saved defaults.'). If confirmed: localStorage.removeItem(STORAGE_KEY), clear all four fixed fields, remove all custom item rows, call updateCostTotal().
- updateCostTotal(): sum title/escrow + warranty + tcFee + all custom item amounts using parseFloat(value) || 0 pattern. Display with toLocaleString('en-US', {minimumFractionDigits: 2, maximumFractionDigits: 2}). Show commission percentage as separate note line.
- addCustomItem(label, amount): creates a custom item row in #custom-items with text input (placeholder "Item name"), number input ($ prefix, placeholder "0"), and remove button. If label/amount provided (from localStorage load), pre-fill values. Attach input event listeners that call saveCostTemplate.
- removeCustomItem(btn): removes the row, calls saveCostTemplate() and updateCostTotal().
- toggleSection(header): toggles .open class on the collapsible content sibling, toggles .rotated class on chevron.
- Attach input event listeners to the four fixed cost fields that call both saveCostTemplate() and updateCostTotal().
